SecondHand Web Project Structure
================================

/admin
    /controllers    - Business logic for admin features
    /layouts        - Admin specific headers/footers
    /*.php          - Admin presentation files (Views)

/php
    /controllers    - Business logic for user features
    /help           - User documentation and help files
    /*.php          - User presentation files (Views)

/config
    /database.php   - Centralised DB configuration and connection

/includes
    /navbar_main.php - Main navigation component
    /navbar_back.php - Secondary navigation component

/chatapp
    /api            - Chat logic and message handling
    /chat.php       - Main chat interface

/server
    /server.js      - Node.js real-time server code (Socket.io)

/assets
    /css            - Main styling files
    /js             - Client-side scripts

/uploads            - User-generated content (images, slips)

index.php           - Root entry point
project_structure.txt - This file
task.md             - Progress tracker
walkthrough.md      - Summary of recent work
